Dota2Bot Privacy Policy

1. Introduction

Dota2Bot ("the Bot") is a Discord bot created to enhance your experience with Dota 2-related features, such as game statistics, hero information, and community interaction. This Privacy Policy explains how the Bot handles user data and ensures compliance with Discord's policies.

2. Information We Collect

Dota2Bot does not collect personal information beyond what is necessary to operate on Discord. The Bot may temporarily access:

  • Discord user IDs, usernames, and discriminators
  • Server and channel IDs
  • Command content (only for processing the command)
  • Bot configuration or settings (if used)

No data is shared, sold, or transmitted outside of Discord unless explicitly stated.

3. How We Use Information

Information is used solely for:

  • Responding to user commands
  • Maintaining server-specific configurations
  • Preventing spam or abuse
  • Improving bot performance

4. Data Retention and Deletion

Dota2Bot does not permanently store message data. Temporary information may be held in memory while a command is being processed and then discarded. Server configurations (if applicable) are stored only as long as necessary to operate features.

You can request deletion of any stored data by contacting the bot owner.

5. Security

Reasonable technical measures are taken to protect data from unauthorized access. However, absolute security cannot be guaranteed.
